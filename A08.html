lista de definición de palabras uso en ámbito laboral.

Variable
Espacio en memoria que guarda un dato que puede cambiar durante la ejecución del programa.
Función
Bloque de código que realiza una tarea específica y puede ser reutilizado.
Condicional
Estructura que permite ejecutar diferentes bloques de código según una condición lógica.
Bucles
Permiten repetir una porción de código varias veces. Ej: for, while.
Python
Lenguaje de programación interpretado, de alto nivel, muy utilizado por su sintaxis clara.
